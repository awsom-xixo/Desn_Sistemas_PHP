Sistema CRUD de Clientes em PHPs

O projeto consiste em um sistema web simples, desenvolvido em PHP, com a funcionalidade de realizar operações básicas de CRUD (Create, Read, Update, Delete) para gerenciar dados de clientes em um banco de dados MySQL.

  • Cadastrar Clientes
  • Pesquisar Clientes
  • Visualizar Clientes
  • Atualizar Clientes
  • Deletar Clientes
